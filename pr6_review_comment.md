# PR #6 レビューコメント

❗ **指摘1（重要）: 数式が欠落しており、仕様が不完全**  
目的関数のタイブレーク定義で、重心距離の式が途中で欠落しています。  
- 最悪コンテナの重心距離 (_k d_k) ...、- (d_k = ) となっており、評価式として成立していません。これだと実装者が同じ評価関数を再現できず、採点基準が曖昧になります。

**提案**  
元の意図（例：max_k d_k や d_k = sqrt((x̄_k-X_c)^2+(ȳ_k-Y_c)^2)）を明記して、式を完全な形に復元してください。

⚠️ **指摘2（軽微）: セクション見出しのMarkdown構造が崩れている**  
4.2 と 7.3 が見出し記法（### など）になっておらず、ドキュメント構造が不統一です。レンダリング時に目次・階層が崩れ、可読性が下がります。

**提案**  
`### 4.2 行先（配送先）制約`、`### 7.3 行先データ（X/Y）` に統一してください。

docx→md変換自体は有益ですが、評価式の欠落は仕様上のブロッカーです。
